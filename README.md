# GIF动画帧提取

utools下的一款简单实用的GIF帧提取工具，分离gif每帧图片插件

> 可以脱离utools 使用，需另行添加复制方法即可



## 👉使用方式：

- 将图片拖进框内，自动分离每一帧
- 点击上传图片自动分离
- 点击单张图片查看复制下载
- 分离的图片全部打包zip下载



## 🧰使用的第三方库：

**vue** + **element-ui** + **libgif-js** + **jszip** + **FileSaver**
